# Минималистичная Hero-секция

![Preview](https://cdn.poehali.dev/templates/minimalist-hero-section-ru/preview-ru.jpg)

Чистая и элегантная hero-секция для лендингов. Градиентный фон от белого к индиго, плавные анимации появления элементов через Framer Motion, блок социального доказательства с аватарами и рейтингом. Идеально подходит для SaaS-продуктов, стартапов и продуктовых страниц.
